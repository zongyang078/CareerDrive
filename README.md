# CareerDrive: Maine Construction Industry Career Map

DS 5230 Unsupervised ML — Final Project (Spring 2026)

## Overview

CareerDrive is a data-driven career mapping tool for Maine's construction industry.
We use O\*NET occupational data, AGC member company data, and education/training program
data to build an interactive career pathway map powered by clustering and network analysis.

This project supports the broader CareerDrive initiative
by MaineDOT and Northeastern University's Roux Institute.

## Setup

### 1. Clone the repo

```bash
git clone <repo-url>
cd careerdrive
```

### 2. Create conda environment

```bash
conda create -n careerdrive python=3.11 -y
conda activate careerdrive
pip install -r requirements.txt
```

### 3. Run notebooks

Open in VS Code or JupyterLab. Start from `01_data_collection.ipynb` and
run in order. Each notebook reads from `data/raw/` and writes to `data/processed/`.

### 4. Run Streamlit demo

```bash
streamlit run app/streamlit_app.py
```

## Data Sources

| Source | Description | Location |
|--------|-------------|----------|
| O\*NET 30.2 Database | Skills, knowledge, abilities per occupation | `data/raw/onet_db/` |
| Client Excel | Companies, apprenticeships, education programs | `data/raw/` |

## Methods

1. **Data Collection**: O\*NET database (local text files) + client Excel
2. **EDA**: Industry landscape — company distribution, occupation categories, education coverage
3. **Clustering**: StandardScaler → PCA → K-Means + DBSCAN on 20 occupations × 120 features
4. **Network Analysis**: Cosine similarity + Related Occupations → centrality → community detection → career map
