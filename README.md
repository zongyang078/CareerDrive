# CareerDrive: Maine Construction Industry Career Map

DS 5230 Unsupervised ML — Final Project (Spring 2026)

## Overview

CareerDrive is a data-driven career mapping tool for Maine's construction industry,
built in collaboration with MaineDOT, AGC Maine, and Northeastern University's Roux Institute.

We analyze O\*NET occupational data, AGC member company data, and education/training program
data using unsupervised learning (clustering, network analysis, topic modeling) to produce
an interactive career pathway map and recommendation prototype.

## Setup

### 1. Clone and install

```bash
git clone <repo-url>
cd careerdrive
python -m venv venv
source venv/bin/activate        # macOS/Linux
# venv\Scripts\activate         # Windows
pip install -r requirements.txt
```

### 2. Configure API (optional)

```bash
cp .env.example .env
# Edit .env with your O*NET API key (get one at https://services.onetcenter.org/)
```

The API is only needed to re-collect text data (notebook 01, section 6).
All processed CSVs are pre-generated in `data/processed/`.

### 3. Run notebooks

Run in order: `01 → 02 → 03 → 04 → 05 → 06`.
Each notebook reads from `data/raw/` or `data/processed/` and writes outputs accordingly.

### 4. Run tests

```bash
pytest tests/
```

Validates that all processed CSVs exist, have the expected shape, and that all mappings cover their source data.

### 5. Run Streamlit demo

```bash
streamlit run app/streamlit_app.py
```

The app can be launched from any directory — paths are resolved relative to the script location.

## Methods

1. **Data Collection**: O\*NET 30.2 database (local) + O\*NET API v2 (text enrichment) + client Excel
2. **EDA**: Company landscape (272 firms), occupation overview, education/apprenticeship coverage, supply-demand gap
3. **Track A — Skills Clustering**: StandardScaler → PCA → K-Means (k=3) + DBSCAN on 20 occupations × 120 features
4. **Track B — Text Clustering**: TF-IDF (200 features) → K-Means + NMF topic modeling on API-enriched occupation texts
5. **Network Analysis**: O\*NET Related Occupations graph → degree/betweenness/PageRank centrality → Louvain community detection
6. **Recommendation Prototype**: Cosine similarity matching — input skills → top-N occupation matches with training pathways

## Key Results

- **3 occupation clusters**: Management/Engineering (Job Zones 4-5), Skilled Trades (Job Zones 2-3), Entry Level/Operators (Job Zone 2)
- **3 NMF topics** align with clusters: Engineering & Management, Installation & Fabrication, Equipment Operations
- **Network hub**: Structural Iron and Steel Workers (degree=10); **Bridge role**: Electricians (betweenness=0.244)
- **Training gap**: Entry Level/Operators cluster has lowest supply-to-demand ratio
